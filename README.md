# emerald-skyline
Nền tảng Web Admin &amp; Mobile App giúp số hóa toàn diện hoạt động quản lý chung cư: quản lý căn hộ, cư dân, phân quyền tài khoản. Tích hợp AI trích xuất chỉ số điện/nước từ ảnh, tự động tạo hóa đơn theo biểu phí linh hoạt. Hỗ trợ thanh toán Momo/VNPay, quản lý phản ánh, tra cứu thông tin và báo cáo trực quan.
